# Scheduling System

A compact Django scheduling system with models, admin registration, serializers, and practical examples.

## Structure

- `models.py` — Core scheduling models
- `admin.py` — Admin registration for models
- `serializers.py` — DRF serializers
- `examples.py` — Example usage patterns
- `requirements.txt` — Python dependencies
- `settings_example.py` — Example Django settings

## Installation

```bash
pip install -r requirements.txt
```

## Usage

- Add the app to `INSTALLED_APPS`
- Run migrations
- Use the Django admin to manage the app
